# Etiqueta Descripción
1 HTML Es el elemento principal de un documento HTML o XHTML, las demás etiquetas deben ir dentro de esta.
2 head Contiene la colección de metadatos acerca del documento, incluyendo enlaces o definiciones de scripts y hojas de estilo.
3 body Esta anotación se encarga de representar el contenido visual de cada documento HTML.
4 script Importa un script desarrollado internamente o apunta un enlace hacia un script externo.
5 section Define una sección dentro del documento HTML. Esta anotación es propia de HTML5.
6 footer Define el pie de página o de una sección.
7 a Sirve para añadir enlaces.
8 svg Define una imagen vectorial embebida.
9 datalist Representa un conjunto de opciones predefiniddas para otros controles.
10 menu Contiene multiples opciones de comando.
# Estilo Descripción Sintaxis
1 color selecciona el color que utlizará el contenido de elemento de texto y decoraciones de texto. color: #009900;
2 background Es una manera de definir los valores del fondo en una única regla CSS. .ejemplo { background: red; }
3 border Define todos los bordes en una única regla de la hoja de estilos. border-ejemplo { border: 2px; }
4 margin Con este estilo se aplicaa margenes en los cuatro lados de la pantalla. margin: 2% auto;
5 font Establece todas las propiedades que utilizaran las fuentes de texto p { font: 12pt sans-serif }
6 height Con este estilo se define la altura de un elemento. height: auto;
7 width Con este estilo se define el ancho de un elemento. width: auto;
8 link representa un elemento que aún no se ha visitado. a:link { color: red; }
9 list-style Permite definir de manera general un estilo a cada elemento de una lista. .ejemplo-list { list-style: circle; }
10 position Define la posición de un elemento dentro del documento. position: static;
